Who is Timey for?

  1. Entrepreneur
  2. Freelancer
  3. Consultant
  4. Intrapreneur
  5. YOU! Anyone who wants to be more efficient and productive with their time.

How do I use timey?

There are several ways you can use Timey. The Timey Team has laid out a few use cases of how some of our customers and the team use the tool. Timey is a tool made by us for you. We encourage you to make Timey your own and use it in a way that makes sense for you. If you think that there are some features missing or if you want to share how you are using timey let us know on twitter @timey.it or by emailing us at feedback@timey.it


How to Create a Task

To create a new Timey task:

  1. Go to the Track Time page
  2. Type in the Name of Task and click the “Create” button.

How to Start Timing a Task

To start timing a task:

  1. Go to the Track Time page
  2. Locate the Task you would like to start timing and select it. (If it is not on the list you can create a new task).
  3. Task name and timer is started at the top of the Track Time page.

How to Stop Timing a Task

To stop timing a task that you have started timing:

  1. Go to the Track Time page
  2. The current task being tracked is displayed at the top of the Track Time page.
  3. Click on “Stop”.

Deleting a Task

To delete a Timey Task:

  1. Go to the Track Time page
  2. Locate the Task that you would like to delete.
  3. Next to the task you will see an X
  4. Click the X to delete the task.
